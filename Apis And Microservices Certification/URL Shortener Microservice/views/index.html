API Project: URL Shortener Microservice

User Story:

  1. POST a URL to https://funky-meteorite.glitch.me/api/shorturl/new and receive a shortened URL in the JSON response.
    Example : {"original_url":"www.google.com","short_url":da37b1}
  2. If an invalid URL did not follow this (http(s)://www.example.com(/more/routes)) format, the JSON response will contain an error like {"error":"invalid URL"}
  3. When you visit the shortened URL, it will redirect you to your original link.

Short URL Creation

example: POST https://funky-meteorite.glitch.me/api/shorturl/new - https://www.google.com
